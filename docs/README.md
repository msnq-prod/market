# Документация ZAGARAMI

В этой папке хранится актуальная документация продукта ZAGARAMI по ролям, эксплуатации и процессам. Внутри репозитория и технических идентификаторов может встречаться внутреннее имя `stones` — это тот же проект.

## Основные документы

- `DOCUMENTATION_PACKAGE_RU.md` — единый master-документ полного пакета документации.
- `BUSINESS_LOGIC_RU.md` — текущая бизнес-логика системы и реальные статусы процессов.
- `BUSINESS_PROCESSES_AUDIT_RU.md` — полное описание системы на языке заказчика и список бизнес-процессов для аудита.
- `BUSINESS_PROCESS_MAP_RU.md` — визуальная карта процессов в виде схем.
- `SYSTEM_USAGE_GUIDE_RU.md` — общее руководство по использованию системы.

## Руководства по ролям

- `USER_GUIDE_ADMIN_RU.md` — работа HQ: приемка, склад, заказы, каталог, пользователи.
- `USER_GUIDE_FRANCHISEE_RU.md` — работа франчайзи: задачи на сбор, партии, финансы.
- `USER_GUIDE_OWNER_RU.md` — управленческий обзор для владельца бизнеса.

## Технические документы

- `ARCHITECTURE.md` — фактическая архитектура приложения и deployment topology.
- `DOCKER_RU.md` — локальный Docker и production stack.
- `OPERATIONS.md` — production deploy, backup, restore, rollback.
- `TEST_CREDENTIALS_AND_TECH_INFO_RU.md` — тестовые креды, seed-данные, ключевые URL и API.
- `RULES.md` — правила развития репозитория и ожидания к изменениям.
- `../VPS_DEPLOY_LOCAL.md` — шаблон локального VPS-runbook без секретов.
